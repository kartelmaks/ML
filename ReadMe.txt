Мой финальный проект в курсе "Пайтон для анализа данных" от МФТИ. 
Классифицировался набор данных fashion_mnist с 10ма наборами одежды. 
Классификация проводилась моделями разной сложности: от самой простой
логистической регрессии к сверточной нейросети с батч-нормамизацией.
Лучший результат на тестовой выборке показала сверточная нейросеть.

Подробный анализ в коде.
